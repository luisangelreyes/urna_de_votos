# urna de votos

Este repositorio está dedicado a un programa simple para saber el numero de votos que obtuvo cada candidato en una urna

En esta ocación, el programa se encarga de contar votos y mostrar en porcentaje el candidato más votado. Cuando se selecione el numero 0, el conteo se termina y muestra el resultado
Para este pequeña solucion se tomó de ejemplo a solo 3 muy importantes pero no tan competentes candidatos:

1. Jorge Alvarez Maynes
2. Xochitl Galvez
3. Claudia Sheinbaum Pardo

teniendo a nuestros contendientes a la grande (la presidencia), utilizé un arreglo con un tamaño de 3, nos servira para asignar los votos a cada candidato y cerrar la casilla  
[0,1,2,3] 
1.Voto para Maynez
2. voto para Xochitl.
3. Voto para claudia
0.Cerrar casilla

El "menú" anterior se mostrará mientras que no se seleccione el numero 0,
Una vez se termina (se elige la opcion 0), se mostrará el total de votos y cuantos votos tuvo cada contendiente, incluyendo sus respectivos porcentajes.
